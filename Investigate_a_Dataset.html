Introduction¶

Here, in this project I am working on the TMDB movies dataset. This dataset contains information about thousand's of movies. It consists of various columns including id,revenue,budget,vote_count etc.
Questions analysed in this project are:

  1. Finding the most popular movies.
  2. Ploting the graph of Year of Release Vs Profitability.
  3. To Find The Average Runtime Of The Movies
  4. No of movies released every year

Conclusions¶

After this analysis we can conclude following:
1.Jurassic World is the most popular movie.
2.Recent movies are getting more profits.
3.Average run time of movies is 109 minutes.
4.Number of movies released each year are increasing.

Limitations¶

1.In my analysis I have removed many columns from the data which according to me was not useful for my analysis but those might be useful in some analysis process.
2.I haven't removed the null values from the genres column. That might affect some results in my analysis. I could have searched the genres of those movies from google.
3.Here,I are not sure if the data provided to us is completely corect and up-to-date.
4.I have just assumed that the budget and revenue of the movies are in dollar, it might be possible that different movies have budget in different currency according to the country they are produce in.In that case the analysis would be incorrect.
